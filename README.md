# Delta VRZ Trading Bot

Automated cryptocurrency futures trading bot for Delta Exchange India using VRZ (Value Rejection Zone) strategy with candlestick pattern recognition.

## Features

- ✅ Multi-timeframe VRZ Support/Resistance detection
- ✅ 20+ Candlestick pattern recognition (2-candle, 3-candle, single-candle)
- ✅ Automated entry signal generation
- ✅ Risk-Reward ratio validation (minimum 1:1.5)
- ✅ Multiple target exits (Zone-based or RR-based)
- ✅ Trailing stop loss after first target
- ✅ Position tracking and management
- ✅ Telegram bot interface for control
- ✅ Real-time monitoring and logging
